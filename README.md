# Lab Report 001 - Basic HTML and CSS Layouts

This repository contains the implementation of a basic web page designed using fundamental HTML5 elements and styled with external CSS3. The project showcases lists, complex table structures, and a student admission form.

## 🚀 Features

### 1. Programming Languages Overview
* An ordered list detailing popular programming languages (JavaScript, Python, C++, Java, Go).
* Nested unordered lists highlighting the primary use case for each language.

### 2. Styled Course Timetable
* A structured schedule utilizing advanced HTML table attributes (`rowspan` and `colspan`).
* Styled dynamically with CSS to feature alternating row background colors (`:nth-child(odd)`) and interactive hover states (`tr:hover`).

### 3. Student Admission Form
* A comprehensive registration form containing various input types: text, email, telephone, date, radio buttons, dropdown select menus, and file uploads.
* Form inputs and buttons are fully responsive and styled with custom margins, borders, and modern box-shadows.
